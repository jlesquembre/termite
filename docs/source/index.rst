=========
 termite
=========

A tool to automate your work.

Features
--------

    - Runs tasks from the command line.

    - Watches for file changes .

    - Serves HTML files and reload them on changes.

Alternatives
------------

A list of alternatives to `Termite`_:

    - `Grunt`_

    - `Brunch`_

    - `Mimosa`_

    - `LiveReload`_



Why another build tool?
-----------------------

After some frustration with the alternatives, I started to write `Termite`_.


Requeriments
------------

`Termite`_ needs `Python <http://python.org>`_ 3.3 or better


Installation
------------

.. code-block:: bash

    pip install termite
